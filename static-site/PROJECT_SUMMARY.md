## \ud83d\udee0\ufe0f Customization Guide

### Change Colors
Edit Tailwind config in each HTML file:
```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                'obsidian': '#0A0A0A',  // Change here
                'signal-red': '#FF3366'  // Change here
            }
        }
    }
}
```

### Add Blog Posts
1. Copy `blog-post.html`
2. Update content, title, date
3. Add card to `blog.html`
4. Update sitemap

### Add Tools
1. Open `tools.html`
2. Copy a tool card
3. Update name, description, link
4. Re-deploy
